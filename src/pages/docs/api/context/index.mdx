# Context

<CodePreview data={contextTypePreview} /> is an externally declared state object declared via <CodePreview data={createContextPreview} />.
The <b>Provider</b> property is a component that acts as a 'host' of the context, that is initialized with it's own value.

Children of the context can use `useContext()` to access the provider's current state - if no provider is found, it will return the context's initial value.

### Example

<div>
  <div className="not-prose">
    <TabGroup
      items={["App", "ThemeContext", "Button"]}
      onSelect={(value) => (selectedTab.value = value)}
      value={selectedTab.value}
      itemSuffix={(item) => `.${siteCodeLang.value}${item === "ThemeContext" ? "" : "x"}`}
    />
    <CopyInnerText>
      <Derive from={{siteCodeLang, selectedTab}}>
        {({siteCodeLang, selectedTab}) => {
          return match(siteCodeLang)
            .with("js", () => { 
              return match(selectedTab)
                .with("App", () => <ExampleApp />)
                .with("ThemeContext", () => <ExampleThemeContext />)
                .else(() => <ExampleButton />)
            })
            .else(() => {
              return match(selectedTab)
                .with("App", () => <ExampleApp />)
                .with("ThemeContext", () => <ExampleThemeContextTS />)
                .else(() => <ExampleButton />)
            })
        }}
      </Derive>
    </CopyInnerText>
  </div>
</div>
